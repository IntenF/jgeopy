# jgeopy

## 概要
このライブラリはpython用リバースジオコーディングです。国内のジオコーディングに限ります。元データには[国土地理院のデータ](http://nlftp.mlit.go.jp/isj/)を使っています。

## 利用法
このjgeopyのファイルをすべて自分のpythonを扱うディレクトリにコピー

'''ex.)
import jgeopy
reverse_geocoding(latitude=35, longitude=135) 
'''
